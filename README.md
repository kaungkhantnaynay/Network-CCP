# Network-CCP

Career Coaching Platform for network and IT professionals.

## Overview

Network-CCP is a React and Vite web app for presenting career coaching,
technical training, program tracks, testimonials, and contact pathways for
network-career candidates.

## Tech Stack

- React
- Vite
- GSAP
- ESLint

## Getting Started

Install dependencies:

```sh
npm install
```

Start the local development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Available Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` creates a production build.
- `npm run lint` runs ESLint across the project.
- `npm run preview` serves the production build locally.
